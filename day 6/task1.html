EMMETS

Emmet is an essential development tool that helps you quickly create repetitive structures like lists, tables, or ordered elements with minimal typing.

It is more like a shorthand that translates to multiple lines of HTML and code or a CSS attribute.

TABLE

Emmets Name Discription
! or html5 For html5 Structure.
By tagname locates elements with a given tag name.
^ it acts the element to the parent. [html>head>title^body]
> child For adding the child tag. [table>tr>td]
+ sibling For adding with the sibling. [table>caption+tr]
* For multiple elements. [table>caption+tr*5>td]
{} The subject or ideas are written,reprasented in content.
attributes For it is used to write the attributes.
